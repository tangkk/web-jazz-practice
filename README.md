# web-jazz-practice

A pure frontend web app for recommending today's jazz practice.

## Goals

- Pure frontend, no backend
- Can work as a simple random picker
- Can also reduce practice decision fatigue
- Supports both personal bias and general-purpose usage
- Source data originates from `JAZZ_PRACTICE.md`
- Intended for GitHub Pages deployment

## Initial scope

- Random recommendation button
- Optional filters: duration / focus / energy / bias
- Smart-enough weighted random selection
- Black / white / gray visual style

## Planned structure

- `index.html` — main app shell
- `styles.css` — visual style
- `app.js` — app logic
- `data/practice-data.js` — structured practice source data
- `docs/` — notes if needed later
